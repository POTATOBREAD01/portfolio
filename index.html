PROJECT

시·도별 전력사용량 예측 대시보드

2025.06 (3인 프로젝트)

본인 작업 내역

  • 프로젝트 설계(조장)
  • 머신러닝 모델 개발
  • 기상청 API활용 전력 사용량 예측 알고리즘 구현
  • 페이지 설계 및 디자인(CSS)
  • 메인 대시보드 페이지 제작
  • 상세 조회 페이지 제작

Deployment URL

https://github.com/POTATOBREAD01/NoName2


프로젝트 목적

시각적 대시보드 제공

전력 사용량 추세나 예측 값을 직관적, 시각적으로 표현하여 비전문가도 쉽게 이해 가능, 의사결정에 도움


전력 사용량 예측

시·도별 전력 사용량을 예측하여 에너지 관리자, 에너지 관련 정책 담당자 등에게 정보 제공


예상 사용자와 이용 목적

예상 사용자

전력 사용량 현황과 예측 데이터가 필요한 사용자


  • 에너지 공기업 (한국전력, 전력거래소 등)
  • 지방자치단체, 정부부처 (환경부, 시·도청 등)
  • 기업, 연구 기관 (에너지 컨설팅 회사, 대학교, 연구소 등)

이용 목적

전력 사용량 현황과 예측 데이터가 필요한 사용자


  • 전력 생산, 공급 계획 수립
  • 지역별 전력 수요 예측을 통한 전력 정책 수립
  • 사업장별 전력 비용 예측

페이지 설계

페이지 이용 목적을 고려한 페이지 설계

  • 메인 페이지 : 현재 날짜를 기준으로 최신 정보 및 단기 예측 데이터를 제공하는 대시보드 페이지
  • 상세 조회 페이지 : 사용자가 원하는 정보를 시각적으로 구성해서 보여주는 대시보드 페이지
  • 다운로드 페이지 : 사용자가 원하는(도시, 연도)를 골라서 엑셀파일, 이미지 등으로 다운로드할 수 있는 페이지

머신러닝 모델 개발

전력 사용량 예측을 위한 머신러닝 모델 개발

데이터 수집 -> 데이터 전처리 -> 모델 테스트 및 최종 알고리즘 선정 -> FastAPI에서 구동 순으로 진행

한국전력공사 홈페이지에서 2014년~2025년 11년차 시·도별 전력 사용량 데이터 수집 후 가공

가공한 11년치의 전력 사용량 데이터를 장기 모델, 단기 모델 훈련에 사용할 수 있도록 전처리

MAE, RMSE, R2 수치를 비교해서 최종 알고리즘 선정

  • 단기 예측 모델 : RandomForest
  • 장기 예측 모델 : RandomForest

RandomForest 장기 모델에서 먼 미래의 데이터 예측시 예측값이 일정값으로 수렴해버리는 문제가 발생,
장기 모델 알고리즘을 시계열 예측에 특화된 Prophet 알고리즘으로 교체

  • 단기 예측 모델 : RandomForest
  • 장기 예측 모델 : Prophet

완성된 모델을 FastAPI 서버에서 구동해서 웹에서 전력 사용량 예측 정보가 필요할 때 URL로 쉽게 접근해서 사용할 수 있도록 함


기상청 API를 활용한 전력 사용량 예측

기상청 API로 받아온 단기예보 정보를 메인 대시보드에 표시 


function predictUsage(maxTemp, weather, month) {
	var isSummer = (month >= 6 && month <= 8);
	var isWinter = (month === 12 || month === 1 || month === 2);

	if (isWinter && weather === '눈') {
		return { text: '증가 예상 ↑', cssClass: 'increase' };
	}
	if (isSummer && maxTemp >= 28) {
	    return { text: '증가 예상 ↑', cssClass: 'increase' };
	}
	if (isWinter && maxTemp <= 5) {
	    return { text: '증가 예상 ↑', cssClass: 'increase' };
	}
	if (weather === '비') {
	    if (isSummer) {
	        return { text: '감소 예상 ↓', cssClass: 'decrease' };
	    }
	    return { text: '유지 예상 →', cssClass: 'neutral' };
	}
	if (maxTemp <= 10) {
	    return { text: '감소 예상 ↓', cssClass: 'decrease' };
	}
	if (maxTemp >= 30) {
	return { text: '증가 예상 ↑', cssClass: 'increase' };
	}

	return { text: '유지 예상 →', cssClass: 'neutral' };
}

기상청 API에서 받아온 날짜, 날씨, 최저기온, 최고기온 데이터로 계절별 기온에 따른 전력 사용량 예측 알고리즘 구현


SVG 파일을 이용한 맵 구현

대한민국 지도를 시·도 별로 클릭해서 상호작용할 수 있도록 SVG파일로 가공

페이지에 맞는 사이즈로 재가공

실제 적용한 모습(포토샵으로 만든 이미지로 맵 위에 각 시·도의 이름을 표시)


프로젝트 결과물

메인 대시보드 페이지

  • 현재 날짜를 기준으로 전력 사용량을 분석 및 예측하는 대시보드 페이지
  • 날씨 데이터를 기반으로 근 시일 내의 전력 사용량 예측
  • 우측의 시·도별 버튼으로 화면에 띄워지는 시·도 변경
  • 그래프 위에 마우스 커서를 올릴 시 상세 값 출력

상세 조회 페이지

  • 사용자가 원하는 연도와 시·도를 골라서 전력 사용량과 전력 사용량 예측 값을 조회하는 대시보드 페이지
  • SVG파일을 이용한 지도로 시·도를 선택
  • 그래프 위에 마우스 커서를 올릴 시 상세 값 출력

다운로드 페이지

  • 각 수치 비교와 전력 정보의 다운로드에 중점을 둔 페이지
  • 사용자가 원하는 연도의 시·도 데이터를 선택해 조회 및 다운로드
  • 다운로드 형식은 csv, xlsx, img 지원
