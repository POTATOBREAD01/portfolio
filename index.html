시·도별 전력사용량 예측 대시보드

2025.07

Deployment URL

https://github.com/POTATOBREAD01


프로젝트 목적

시각적 대시보드 제공

전력 사용량 추세나 예측 값을 직관적, 시각적으로 표현하여 비전문가도 쉽게 이해 가능, 의사결정에 도움


전력 사용량 예측

시·도별 전력 사용량을 예측하여 에너지 관리자, 에너지 관련 정책 담당자 등에게 정보 제공


예상 사용자와 이용 목적

예상 사용자

전력 사용량 현황과 예측 데이터가 필요한 사용자


  • 에너지 공기업 (한국전력, 전력거래소 등)
  • 지방자치단체, 정부부처 (환경부, 시·도청 등)
  • 기업, 연구 기관 (에너지 컨설팅 회사, 대학교, 연구소 등)

이용 목적

전력 사용량 현황과 예측 데이터가 필요한 사용자


  • 전력 생산, 공급 계획 수립
  • 지역별 전력 수요 예측을 통한 전력 정책 수립
  • 사업장별 전력 비용 예측

페이지 설계

머신러닝 모델 개발

기상청 API를 활용한 전력 사용량 예측

SVG 파일을 이용한 맵 구현
